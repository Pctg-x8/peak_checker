# peak_checker
Peak Checker of waves in the folder<br>
音楽ゲームの製作時に有用(かもしれない)なツールです。<br>
同一フォルダ内(サブフォルダは含まれません)にあるwavのピークを計測し、<br>
様々な統計データをファイルに出力します。<br>

Usage
---

> peak_checker [フォルダパス(省略すると現在のフォルダ)]

ダブルクリックして起動するとexeがあるフォルダに対して処理を行います。
